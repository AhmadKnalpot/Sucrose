# Sucrose
 Sucrose is a Smart Agriculture IoT project designed to improve melon cultivation through automatic irrigation and AI-based ripeness detection. The irrigation system uses an ESP32, Soil Moisture sensor, relay, water pump, OLED display, Blynk, and a PHP-MySQL web dashboard for real-time monitoring. The ripeness detection feature utilizes an Edge Impulse AI model to classify melon maturity (Unripe, Half-Ripe, and Ripe) using a smartphone camera via QR Code.
